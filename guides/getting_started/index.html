    Getting Started

    Overview

    Dopus is a framework for building AI agents and tools on top of existing LLM providers. It doesn't include any agents or tools itself, but rather a fast way to build agents and tools yourself.

    Installation

    pip install dopus

    Let's also install python-dotenv and openai:

    pip install python-dotenv openai

    Setup

    Create a .env file with your OpenAI API key:

    OPENAI_API_KEY={YOUR_API_KEY_HERE}

    Hello World Agent

    Let's create a simple Hello World agent that uses tools and interacts with the user. 

    import dotenv
dotenv.load_dotenv()

from dopus.core import Agent, tool

class HelloAgent(Agent):

    @tool(
        description="Greet the user",
        parameters={
            "name": {
                "type": "string",
                "description": "Name of the user to greet"
            }
        },
        required=["name"]
    )
    def greet(self, name):
        print(f"Agent: Hello, {name}! Nice to meet you.")

    @tool(
        description="Ask the user a question and get their response",
        parameters={
            "question": {
                "type": "string",
                "description": "The question to ask the user"
            }
        },
        required=["question"]
    )
    def ask_user(self, question):
        print(question)
        user_response = input("User: ")
        return user_response

    @tool(description="End the conversation")
    def done(self):
        print("Agent: Goodbye!")
        self.stop()

    def prompt(self):
        return "You are a friendly AI that greets users. You ask their name and greet them."

agent = HelloAgent()
agent.run()
